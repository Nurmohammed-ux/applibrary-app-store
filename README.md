# Hero App Store

## 📱 App Name

**Hero App Store**

---

## 📖 Description

Hero App Store is a modern web application that showcases trending mobile applications. Users can browse different apps, view details such as downloads, ratings, reviews, and app size. The project is designed to simulate an app marketplace similar to popular app stores.

The goal of this project is to practice building reusable components, managing structured JSON data, and displaying dynamic content using React.

## 🚀 Technologies Used

* **React** – For building the user interface
* **Vite** – Fast development environment and bundler
* **JavaScript (ES6)** – Core programming language
* **Tailwind CSS / CSS** – Styling the UI
* **Lucide React** – Icon library for modern UI icons
* **JSON Data** – Mock data for app information

---

## ⚙️ Features

* Display trending apps
* Show app downloads and ratings
* App cards with image, title, and stats
* Responsive UI
* Icon support using Lucide React
